Aplikasi mobile presensi pengajian yang dikembangkan sebagai project Ujian Akhir Semester
mata kuliah Pengembangan Aplikasi Mobile.

## Deskripsi Singkat
Aplikasi ini digunakan oleh pengurus pengajian untuk melakukan presensi jamaah secara digital
dengan sistem kontrol terpusat. Presensi dilakukan berdasarkan foto dan nama jamaah, tanpa
melibatkan presensi mandiri oleh jamaah.

## Fitur Utama
- Login pengurus
- Presensi harian jamaah (Hadir / Izin / Tidak Hadir)
- Manajemen data jamaah
- Pengambilan foto jamaah menggunakan kamera
- Rekap dan statistik kehadiran

## Teknologi (Planned)
- Flutter (Mobile App)
- Supabase (Database & Storage)
- Kamera perangkat (Sensor)
