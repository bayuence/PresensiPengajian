# Presensi Pengajian – Mobile App

Aplikasi mobile presensi pengajian yang dikembangkan sebagai proyek
Ujian Akhir Semester (UAS) mata kuliah Pengembangan Aplikasi Mobile.

Aplikasi ini dirancang untuk membantu pengurus pengajian dalam
mengelola kehadiran jamaah secara digital dengan sistem kontrol terpusat,
tanpa melibatkan presensi mandiri oleh jamaah.

---

## Deskripsi Aplikasi

Presensi dilakukan sepenuhnya oleh pengurus dengan melihat daftar jamaah
berdasarkan nama dan foto. Pengajian tidak bersifat harian, melainkan
dibuat secara fleksibel ketika kegiatan pengajian berlangsung.

Dalam satu hari dimungkinkan terdapat lebih dari satu kegiatan pengajian.
Setiap kegiatan presensi dapat dibuka dan ditutup secara manual oleh
pengurus, serta akan ditutup otomatis pada akhir hari apabila belum
ditutup.

---

## Fitur Utama

- Login pengurus
- Pembuatan dan pengelolaan presensi pengajian
- Presensi jamaah (Hadir / Izin / Tidak Hadir)
- Manajemen data jamaah (tambah, ubah, hapus)
- Pengambilan dan penyimpanan foto jamaah
- Rekap dan laporan kehadiran jamaah

---

## Arsitektur Aplikasi

Aplikasi dikembangkan menggunakan pendekatan **Model-View-Controller (MVC)**:
- **Model**: Representasi data jamaah, pengajian, dan presensi
- **View**: Antarmuka pengguna (Flutter UI)
- **Controller**: Pengelolaan logika aplikasi dan alur presensi

---

## Teknologi yang Digunakan

- Flutter (Mobile Application)
- MySQL (Database)
- Backend API (planned)
- Kamera perangkat (pengambilan foto jamaah)
